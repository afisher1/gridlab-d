# Random numbers - GridLAB-D Wiki

**Source URL:** https://gridlab-d.shoutwiki.com/wiki/Random_numbers
Random numbers may be generated by one of the following supported distributions: 

  * [bernoulli]
  * [beta]
  * [degenerate]
  * [exponential]
  * [gamma]
  * [lognormal]
  * [normal]
  * [pareto]
  * [rayleigh]
  * [samples]
  * [triangle]
  * [uniform]
  * weibull
